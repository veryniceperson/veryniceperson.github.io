Step 1: Writing the HTML code for your website

In order to start creating your website you would need to learn how to code in HTML. In this step you will learn basic HTML, in order to write all of the content of your website.
Let's get started!

Parts of HTML code

In the image to the right you would be able to see a diagram that represents the parts of a single element. The element is a part of code in HTML which consists of an HTML tag, the content inside and sometimes some styling.

HTML tag:
The HTML tag contains the element name and helps classify a piece of content, because without the tag the browser wouldn't know how to handle the content inside the tags. The tag starts with "<" then has the class and then ends with ">".

The opening tag:
The opening tag is the first HTML tag and is utilised to commence an HTML element. It starts with a "<" has the class and then ends with a ">".

The content:
The content inside of the the HTML opening and closing tags is the information that you add for the website to contain. It comes in many forms including text, images, videos, audio clips and other types of files.

The closing tag:
The closing tag is the ending HTML tag in a HTML element. The difference between the opening tag and the closing tag is that the closing tag has a "/" after the "<". The are used to mark the end of an element in a website.

After you have read through this and memorised it, we can start learning the code structure (hierarchy) in order to make sure that your website code is organised and easy to edit.

HTML Hierarchy (HTML code structuring)

In the image above you would be able to see how HTML hierarchy works. It starts with parent, then goes on with children, siblings, grandchildren and so on. It is important to understand HTML Hierarchy because the decendants of different elements can inherit properties such as text colour and font from their ancestors.

Getting Started

Now we are going to start writing some code! So get your keybooard ready and let's get started!

First of all we are going to need a platform on which we would be able to write our code on. You can either use your own, something like Atom or Visual Studio. But if you are unable to do so, use a Online HTML and CSS editor.

Starting off on a black page
Now, erase anything that is present in your HTML editor. Then on top of your document write <!DOCTYPE html> . This will allow the user's browser to know what type of file it is and how to deal with the file. This should be included in all of your html documents.

The HTML declaration
In a new line, write <html> . This is the parent of all of the element and will contain all of the content in the document, except for <!DOCTYPE html> . It is important that you include this declaration in all of your html documents.

The <head> tag
In another line type in <head> . This element contains content that is usually not displayed on the webpage itself and contains the metadata of the website. Metadata is stuff like styling, the title of the website that you see on the tab of your browser, the image that is displayed on the browser tab whenever the website is open and the link to the stylesheet of the website (We will get into detail about that in Step 2) .

The title on the tab of a browser
Now, type in <title> and </title> . This would allow you to include a title on the tab of a browser for your website. This is optional but, if you don't include a title, the website's name on the tab would appear as Untitled (You should include the title text in between the opening and closing tag).

If you want to include a website icon, you can include this code after the <title> : <link rel="shortcut icon" href="Insert image link here" /> This is optional, but highly recommended because the website icon on the tab of the browser adds a nice touch and makes your website's tab more distinguishable compared to other tabs.

Stylesheet link
Last, but not least in this list, we have to include a link to our stylesheet. The stylesheet is where you would usually write code to give your different properties such as different fonts, although you could write it in the html document itself. This is optional, but it is considered necessary for all websites, because without styling your website would be plain and ugly. On a new line insert <link rel="stylesheet" href="insert link here"> . Now you are free to enter which ever stylesheet you would want to use.

Well done on coming this far!!! Now all you have to do is close the head tag (if you don't remember how to do this you just write </head> on a line below the second link element) and do 2 things to finish your started template.

The body element
Next we have to include the body elememt. The body element is what will contain all of the content of the website, such as text, images, videos and other types of content. First try to write the code for the tags by yourself and remember, the tag is called body. If you are stuck, use <body> for the opening tag and </body> for the closing tag. In between of those two, you will include all of the visible content of the website.

Finally you would need to close off your html tag. Try to do this yourself, but if you are stuck just paste </html> after the closing body tag.

Paragraphs and Line Breaks

In this section you will learn how to use paragraphs and line breaks. in your website. Paragraphs are useful for adding information about a topic on your website. Paragraphs are usually located below the heading or sub-heading of you website. In order to include a paragraph in your website, you would have to include <p> as your opening tag and </p> as your closing tag. Then, you would include your content in between of those two tags. In a website you would usually have to include your <p> element under a subheading.

In addition, according to the spaces you have in between your elements in the HTML file, is the amount of space you would have in between different groups of content in your website. For example, if you have 1 empty line between a h2 element and a p element in the HTML file, you would have 1 empty line between the heading and the paragraph in the website. However, this doesn't work with different groups of text in one p element and you would have to include breaks.

One way you could space your paragraph in your website is by having many different p elements. A new p element allows elements to have a double line break in between them, as if you had pressed the enter key. Another method is by having only one pair of p tags under a heading and including line breaks in order to seperates paragraphs. The advantage of this is that this method is a bit faster than including new p tags everytime and you would be more organised as you would have only one p tag under a heading. The disadvantage of this method is that sometimes you would need to include more than 1 line break, which could confuse some people as they would not know how many to include exactly. You would see this method being used in the first image.

Line breaks are useful in sepreating content and adding more space between them. Although this is possible to do by adding empty lines in the HTML file itself, sometimes it is best to use line break instead of loads of spaces.

Take this situation in this image for example. In the first image you can see that there are lots of empty line in order to create some space between the two img elements. This makes the file unorganised and you could delete the spaces if you forget what they are for, which could ruin your website. In the second image you can see that with using line breaks, we have a much more organised file and you are less likely to mess up your website by removing the spaces. I recommend using line breaks, because you have more freedom to choose where to put a line break and how many line breaks.

On top of this paragraph you would be able to see the result of the two. The result in both of these codes are the same, the only difference is the structuring and the usage of line breaks. As you would be able to see, the example without any line breaks needs lots of empty lines in order to have one space. This can also be done with only one empty line between the 2 elements, but the extra lines demonstrate that no matter how many empty lines you have, if there is one empty line between two p elements, only one spacing will be showing in the web browser. This is why I recommend using line breaks, because you can customise your webiste whilst keeping the HTML file organised and easy to understand and edit.

You should also remember that it doesn't matter if you close off the line break or not. Either the closing or the opening tag would work as line breaks for your website and if you open a br element, you don't need to close it.

Images

In this section you will learn how to use images in your website. Images are important for you website, as they make it easy for your viewer to understand the context of your website. In addition, images are useful for grabbing the visitor's attention, so if you would want to make your website popular, it is recommended that you use images. However, you must make sure that your images are organised and not all over the place, or your website would look bad and unprofessional.

In order to add an image to your website, you must use the <img> tag. There is no closing tag for the img element, so you just need to opening tag. Your img element must have a source. You include a source for the image by writing src="Insert link here" inside the img tag. Remeber, the scr has the be after the img part of the element and before the closing angle bracket ">".

For now, the images looks a bit too big. The size of the image by default depends on the size of the image originally. However, we can adjust this very easily.

If you would like to adjust the width of the image, you can do so by including this propertiy inside the img tag: width="". The value can be in either px or %. The px value is in pixels and will stay the same on all screen sizes, but the percentage value will adjust the size of the image according to the screen size.
For example, if you set the width of an image to 100%, the image width would take 100% of the screen width, no matter what. However, if you set it too 100px, the image width would only be 100px on all screens, meaning that it would be too small for some screens and too big for others.
